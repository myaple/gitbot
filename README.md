## Getting Started

### Building

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd gitbot
    ```
2.  Build the project:
    ```bash
    cargo build
    ```
    For a release build:
    ```bash
    cargo build --release
    ```
    The executable will be located at `target/debug/gitbot` or `target/release/gitbot`.

### Running

To run the bot:
```bash
./target/debug/gitbot
```
